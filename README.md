# ✈️ 航班反向票購買管理系統 (Reverse Ticket Manager)

這是一個純前端的靜態 React 網頁應用程式，專為喜愛利用「反向機票」策略節省費用，或是頻繁往返兩地並需要整合「機票、住宿與活動票卷」複雜邏輯的進階旅客所設計。

系統具備強大的趟次拆解演算法，能將您東拼西湊的多張「單程票」、「正反來回機票」自動重構，並依照時間軸重新配對成您**實際出門到回家**的飛行「趟次 (Trips)」。搭配住宿與活動管理，幫助您無痛防呆，清楚預覽每一趟旅程的總成本與時間線。

🔗 **[線上體驗 Demo](https://imhahac.github.io/reverse-ticket-manager/)**

---

## 🌟 核心功能說明

- 🎟️ **智慧趟次配對演算**：輸入任意機票，系統以台灣機場為基準，自動為您拆解並組合為合理的「出國到回國」趟次。完美支援外站轉機、不同點進出 (Open-Jaw) 與跨日紅眼航班自動修正。
- 📊 **預算分析儀表板 (Dashboard)**：首頁自動將各類別 (機票/住宿/活動) 花費占比視覺化，並支援多幣別即時匯率換算，精準計算出每趟旅程的「每日平均花費 (CP值)」。
- 🗺️ **跨維度行程防呆 (地圖與排程校驗)**：結合航班到達時間與住宿預定的真實地理位置。當系統發現您有「地點落差巨大」(例如降落關西，卻訂了關東飯店) 或「活動時間衝突」時，會立刻在地圖與列表上給予紅燈警告。
- ☁️ **全客戶端無縫 Google 整合**：
  - **Drive 自動備份**：完全 100% Client-Side 運行，所有紀錄存於瀏覽器 LocalStorage。但您可以透過 Google 登入，一鍵打包 JSON 上傳至您私人的 Google Drive 做備援。
  - **Calendar 行事曆推播**：點擊同步後，會將您的航班轉為具備時區設定的精確事件，住宿則自動轉為全天連續事件，完美對接個人行事曆。
